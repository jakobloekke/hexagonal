# @secondgen/ui

Shared UI component library for the SecondGen Platform.

## Consumption

### From Monorepo (Apps)

```ts
import { Button } from '@secondgen/ui';
```

### External Usage (Replit/Prototyping)

This package is designed to be easily copy-pasteable.

1. Copy `packages/ui/src/components/[Component]` to your project.
2. Ensure you have `tailwindcss` configured.
3. If you need the full library built:
   - Run `pnpm build` in this directory.
   - Use the `dist/` folder artifacts.
